# Naruto Nexus

Naruto Nexus is a web application that serves as the ultimate destination for everything related to the Naruto series. From detailed character profiles to intricate family connections, explore the hidden secrets of the Hidden Leaf Village. Whether you're a seasoned fan or new to the series, begin your journey today and immerse yourself in the captivating world of Naruto!

## Features

- Discover detailed character profiles with images, jutsu, nature types, personal information, and more.
- Explore village profiles and learn about their history, notable members, and significant events.
- Search for characters, villages, clans, kekkei genkai, tailed beasts, teams, and more.
- Random feature to explore a random character from the Naruto universe.
- User-friendly interface with intuitive navigation and responsive design.

## Technologies Used

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- External APIs: [Dattebayo API](https://dattebayo-api.onrender.com)

## Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/naruto-nexus.git
   ```

2. Install dependencies:

   ```bash
   cd naruto-nexus
   npm install
   ```

3. Start the server:

   ```bash
   npm start
